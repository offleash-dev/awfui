# awfui
AWFUI — A lightweight embedded UI framework for C++.
MIT licensed. Designed for clarity, portability, and small systems.

AWFUI (think “awww… fooey”) is a lightweight, embedded‑friendly UI framework built on top of **Adafruit_GFX**. Using intentionally minimal C++, It provides a consistent widget model, a screen/state system, dialogs and modal overlays, and a unified event model.

While based on Adafruit_GFX, AWFUI also includes an SDL-based backend to simplify desktop development and rapid prototyping.
